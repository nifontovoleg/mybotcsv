## Telegram-бот для командной работы с задачами

Простой учебный проект на Python и **aiogram 3**, который показывает:

- **как сделать Telegram-бота**;
- **как работать с SQLite3**;
- **как хранить команды, обработчики, клавиатуры и базу данных по разным файлам**.

### Структура проекта

- **`main.py`** — точка входа, запуск бота и регистрация роутеров.
- **`app/config.py`** — получение токена бота из переменной окружения.
- **`app/db/database.py`** — создание таблицы `tasks` и функции для работы с задачами.
- **`app/handlers/start.py`** — команда `/start`.
- **`app/handlers/tasks.py`** — команды `/add`, `/list`, `/list_csv`.
- **`app/keyboards/main_menu.py`** — простая клавиатура с основными командами.

### Установка

1. Установите Python **3.10+**.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Создайте переменную окружения с токеном бота:

```bash
# PowerShell (Windows)
$env:BOT_TOKEN="ВАШ_ТОКЕН_БОТА"

# или временно для одной команды:
setx BOT_TOKEN "ВАШ_ТОКЕН_БОТА"
```

### Запуск бота

```bash
python main.py
```

### Команды бота

- **`/start`** — приветствие и инициализация базы данных.
- **`/add`** — запустить добавление новой задачи (бот попросит ввести текст отдельным сообщением).
- **`/list`** — показать все задачи в виде текста.
- **`/list_csv`** — отправить CSV-файл `tasks.csv` со всеми задачами.
- **`/delete`** — начать удаление задачи (бот попросит ввести номер задачи из списка `/list`).

База данных хранится в файле **`tasks.db`** в корне проекта.
